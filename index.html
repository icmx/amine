Inline and block

All the input elements are inline by default, but able to be a block by using .am-(input|select|textarea)--block:

Validation

All the boxed inputs supports validation. There are dynamic validation (for :valid and :invalid element states) and static validation (for server-side or some framework-oriented implementations).

For dynamic validation, one can use .am-(input|select|textarea)--validable and .am-...--invalidable classes.

In turn, for static validation there are .am-(input|select|textarea)--valid and .am-...--invalid as well.
